# Project Memory

## Core
Banking SBL Portal. Enterprise compact density (80%-zoom feel at 100%). 14px base, 31px control height.
Primary #2B5797 (215 60% 38%). Radius 6px. Neutral minimal style. No flashy design.
Sidebar fixed 14rem, hamburger overlay on mobile (<768px). AppLayout wraps all pages.
Form grid: 3-column (repeat(3,1fr)) on desktop, 2-col tablet, 1-col mobile. Inter font family.
Required fields use red pill badge "required" not asterisk. Disabled fields get muted bg + no border.
Info tooltips on technical fields (DOE, ESRR, SBS, Exposure). Sticky form actions at bottom.
